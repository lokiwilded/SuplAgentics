# @ramtinj95/opencode-tokenscope

**Type:** npm OpenCode plugin  
**Package:** `@ramtinj95/opencode-tokenscope`  
**Installed at:** `~/.config/opencode/node_modules/@ramtinj95/opencode-tokenscope/`

## What it does

Tracks token usage per tool call and session, providing real-time visibility into which operations are most expensive. Helps identify and optimise high-token workflows.

## Notes

- No special config file required
- Metrics visible in OpenCode's built-in status bar / session info
- Complements the Usage page in the suplagentics dashboard (which aggregates totals from SQLite)
