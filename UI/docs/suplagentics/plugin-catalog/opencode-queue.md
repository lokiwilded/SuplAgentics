# opencode-queue

**Type:** npm OpenCode plugin  
**Package:** `opencode-queue` v0.10.1  
**Installed at:** `~/.config/opencode/node_modules/opencode-queue/`

## What it does

Provides a task queue for OpenCode sessions — lets you queue up multiple prompts/tasks and have them processed sequentially without waiting for each one to finish before sending the next.

## Notes

- Useful when you want to fire off a batch of independent coding tasks
- Integrates with OpenCode's session management
- No special config file required
